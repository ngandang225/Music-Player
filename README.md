# Music-Player
Make a simple Music Player using HTML, CSS, JavaScript

Features:
1. Render songs
2. Scroll top
3. Play / Pause / Seek
4. CD rotate
5. Next / Prev
6. Random
7. Next / Repeat when ended
8. Active song
9. Scroll active song into view
10. Play song when click
 * 
11. Speed
12. Display duration and ... (thời lượng bài hát và thời lượng đã phát)
